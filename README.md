# Bakery Data Wrangling Project

## Objective
Clean and prepare bakery transaction dataset for analysis.

## Dataset
- 20,507 records
- 5 original columns
- 10 columns after feature engineering

## Data Cleaning Steps
- Removed duplicates
- Converted DateTime column to proper datetime format
- Extracted Date, Time, Year, Month, and Hour
- Standardized text columns

## Output
Final cleaned dataset saved as bakery_cleaned.csv
